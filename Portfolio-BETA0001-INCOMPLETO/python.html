Visão Geral

Aplicação web construída com Python e Flask que realiza conversão de moeda em tempo real. O usuário informa um valor em Reais (BRL) e seleciona a moeda alvo — o sistema aplica as taxas de câmbio e exibe o resultado instantaneamente.

🐍 O back-end usa Flask para roteamento e processamento; a lógica de conversão é replicada em JS para a demo interativa abaixo.

1. Estrutura do App Flask

app.py
Python 
from flask import Flask, render_template, request

app = Flask(__name__)

# Taxas de câmbio fixas (BRL como base)
TAXAS = {
    'usd': 5.20,
    'eur': 5.50,
    'gbp': 6.00
}

def converter_moeda(valor_brl: float, moeda: str):
    """Converte BRL para a moeda desejada."""
    if moeda in TAXAS:
        resultado = valor_brl / TAXAS[moeda]
        return round(resultado, 2), moeda.upper()
    return None, None

@app.route('/', methods=['GET', 'POST'])
def index():
    resultado = None
    if request.method == 'POST':
        valor = float(request.form['valor'])
        moeda = request.form['moeda']
        resultado, simbolo = converter_moeda(valor, moeda)
    return render_template('index.html', resultado=resultado)

A função converter_moeda recebe o valor em BRL e o código da moeda alvo, dividindo pelo câmbio fixo configurado no dicionário TAXAS.
