Visão Geral

Implementação completa de uma pilha (stack) em C, usando alocação dinâmica de memória. A estrutura armazena relatórios com ID, título e data. Operações: push, pop, peek, listar, contar e buscar por título.

⚙️ Estrutura LIFO (Last In, First Out) — o último elemento inserido é o primeiro a ser removido. Usa ponteiros e malloc/free para gerenciar memória dinamicamente.

1. Definição da Estrutura

pilha.c
C 
// Estrutura que representa um nó da pilha
struct Relatorio {
    int              id;
    char             titulo[50];
    char             data[20];
    struct Relatorio *prox;  // ponteiro para o próximo nó
};

// Ponteiro global para o topo da pilha
struct Relatorio *topo = NULL;
int proximo_id = 1;

2. Push — Inserir

pilha.c — push_relatorio()
C 
void push_relatorio(const char *titulo, const char *data) {
    // Aloca memória para o novo nó
    struct Relatorio *novo = malloc(sizeof(struct Relatorio));
    if (!novo) { printf("Erro: sem memória\n"); return; }

    // Preenche os campos
    strncpy(novo->titulo, titulo, 49);
    strncpy(novo->data, data, 19);
    novo->id = proximo_id++;

    // Insere no topo: novo aponta para o topo atual
    novo->prox = topo;
    topo = novo;  // topo agora é o novo nó
}

3. Pop — Remover

pilha.c — pop_relatorio()
C 
void pop_relatorio() {
    if (!topo) {
        printf("Pilha vazia!\n");
        return;
    }
    // Guarda ponteiro para liberar memória
    struct Relatorio *temp = topo;
    topo = topo->prox;  // avança o topo
    free(temp);          // libera memória
}

4. Peek, Listar & Buscar

pilha.c — outras operações
C 
// Peek: ver o topo sem remover
struct Relatorio* peek() {
    return topo;
}

// Contar elementos
int contar() {
    int c = 0;
    struct Relatorio* atual = topo;
    while (atual) { c++; atual = atual->prox; }
    return c;
}

// Buscar por título
struct Relatorio* buscar(const char* titulo) {
    struct Relatorio* atual = topo;
    while (atual) {
        if (strcmp(atual->titulo, titulo) == 0)
            return atual;
        atual = atual->prox;
    }
    return NULL;
}
