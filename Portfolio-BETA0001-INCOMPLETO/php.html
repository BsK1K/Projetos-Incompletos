Visão Geral

Sistema completo de gerenciamento de produtos usando PHP no back-end e MySQL para persistência. Permite adicionar, listar, atualizar e deletar produtos por categoria e loja — as quatro operações do CRUD.

CREATE READ UPDATE DELETE

🐘 PHP server-side processa formulários POST; MySQL armazena os produtos. A demo abaixo simula toda a lógica em JavaScript.

1. Conexão com Banco

config.php
PHP 
<?php
// Configuração da conexão PDO
$host = 'localhost';
$db   = 'lista_compras';
$user = 'root';
$pass = '';

try {
    $pdo = new PDO(
        "mysql:host=$host;dbname=$db;charset=utf8",
        $user, $pass,
        [PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]
    );
} catch (PDOException $e) {
    die("Conexão falhou: " . $e->getMessage());
}
?>

2. CREATE — Inserir Produto

produto.php — create
PHP 
<?php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $nome      = htmlspecialchars($_POST['nome']);
    $categoria = htmlspecialchars($_POST['categoria']);
    $loja      = htmlspecialchars($_POST['loja']);
    $qtd       = (int) $_POST['quantidade'];

    // Prepared statement — previne SQL injection
    $stmt = $pdo->prepare(
        "INSERT INTO produtos (nome, categoria, loja, quantidade)
         VALUES (:nome, :categoria, :loja, :qtd)"
    );
    $stmt->execute([
        ':nome' => $nome, ':categoria' => $categoria,
        ':loja' => $loja,  ':qtd'       => $qtd
    ]);
}
?>

3. READ & DELETE — Listar e Remover

produto.php — read / delete
PHP 
<?php
// READ — buscar todos os produtos
$produtos = $pdo->query("SELECT * FROM produtos ORDER BY id DESC")
                ->fetchAll(PDO::FETCH_ASSOC);

// DELETE — remover por ID
if (isset($_GET['delete'])) {
    $id = (int) $_GET['delete'];
    $pdo->prepare("DELETE FROM produtos WHERE id = ?")
        ->execute([$id]);
    header("Location: index.php");
    exit();
}
?>

<!-- Template PHP para listar -->
<?php foreach ($produtos as $p): ?>
  <li>$p['nome'] ?> — $p['categoria'] ?></li>
<?php endforeach; ?>
