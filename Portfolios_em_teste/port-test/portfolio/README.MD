4️⃣ Como Rodar Localmente
1 Salve os 3 arquivos nas pastas indicadas
2 Abra o arquivo index.html diretamente no seu navegador (duplo clique)
3 ✨ Pronto! Seu portfolio está funcionando
    💡 Dica: Para testar as animações de scroll, use a rolagem do mouse. Para ver o efeito do cursor personalizado, mova o mouse sobre links e cards.
